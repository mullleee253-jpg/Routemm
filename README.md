# Routeem - Безопасный мессенджер

Современный мессенджер с акцентом на безопасность и конфиденциальность.

## Возможности

✅ **Авторизация** - Регистрация и вход с реальной проверкой
✅ **Чаты** - Обмен сообщениями в реальном времени
✅ **Звонки** - История звонков и возможность совершать новые
✅ **Настройки безопасности** - Двухфакторная аутентификация, защита от DDoS
✅ **Конфиденциальность** - Настройки приватности и видимости
✅ **Плавный интерфейс** - Белый дизайн с анимациями

## Установка

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Технологии

- Next.js 14 (React)
- TypeScript
- CSS Modules
- Framer Motion (анимации)

## Безопасность

- Сквозное шифрование сообщений
- Защита от DDoS атак
- Двухфакторная аутентификация
- Блокировка подозрительных входов
- Настройки конфиденциальности

## Структура проекта

```
app/
  ├── api/auth/          # API авторизации
  ├── layout.tsx         # Основной layout
  ├── page.tsx           # Главная страница
  └── globals.css        # Глобальные стили

components/
  ├── Auth.tsx           # Компонент авторизации
  ├── MainApp.tsx        # Основное приложение
  ├── ChatList.tsx       # Список чатов
  ├── ChatWindow.tsx     # Окно чата
  ├── Calls.tsx          # Звонки
  └── Settings.tsx       # Настройки
```

## Разработка

Для продакшена необходимо:
- Подключить реальную базу данных (PostgreSQL, MongoDB)
- Настроить WebSocket для реального времени (Socket.io)
- Добавить хеширование паролей (bcrypt)
- Настроить JWT токены
- Добавить WebRTC для видео/аудио звонков
- Настроить SSL сертификаты
